## 题一
题目：Generate Parentheses，见 LeetCode 22。输入 n，输出所有 n 个左右括号的合法括号序列。

**分析**：

+ 关键：当前位置左括号和右括号树 (x, y)(x >= y)
+ 图在哪里？
  + 节点：目前位置左括号和右括号数 (x, y)(x >= y)
  + 边：从 (x, y) 到 (x + 1, y) 和 (x, y + 1)
  + x == y 时，没有 (x, y + 1) 这条边
+ 解是什么？
  + 从 (0, 0) 出发到 (n, n) 的全部路径

**方法**：
+ DFS：
  + 左右括号的个数
  + 当前的“部分解”
+ BFS：
  + 方法 1：当前的“部分解”——比较浪费空间
  + 方法 2：每个节点记录能到达它之间的节点集合——相对比较麻烦，且需要还原路径
